# 撸车日记小程序端

原生微信小程序 + TypeScript，直接对接 `backend` 已实现的 `/api/v1` 接口。

## 运行

1. 用微信开发者工具导入 `miniprogram` 目录。
2. 本地调试时在微信开发者工具里关闭“校验合法域名”。
3. 默认 API 地址在 `utils/config.ts`：`http://127.0.0.1:8080/api/v1`。
4. 如果用真机预览，把地址改成电脑局域网 IP，例如 `http://192.168.1.20:8080/api/v1`。

## 已包含页面

- 首页：帖子列表、类型筛选、搜索、详情入口。
- 二手件：出售/求购列表、分类筛选、搜索、详情入口。
- 发布：发布帖子、出售件、求购件。
- 我的：开发登录、资料编辑、手机号绑定、车库、我的发布和意向。
- 详情页：帖子评论/点赞/收藏/举报，二手件收藏/交易意向/举报。

后端当前使用 `X-User-ID` 作为开发态用户身份，小程序登录后会自动保存并在请求头里携带。
