# 改装知识库 · 无限分类大纲

> 本文档面向“改装车案例分享社区 + 二手件交易撮合平台”中的**知识库模块**，采用无限分类（多级分类）结构，适配小程序逐层浏览体验。  
> 版本：v1.0  
> 更新日期：2026-07-12

---

## 一、什么是“无限分类”模式？

无限分类（又称多级分类）是指每个分类下可以无限创建子分类，没有层级深度限制。例如：
改装知识
├── 动力系统
│ ├── 进气
│ │ ├── 高流量风格
│ │ │ ├── 原理
│ │ │ ├── 品牌
│ │ │ └── 如何选择
│ │ ├── 冬菇头
│ │ └── 进气管道
│ ├── 排气
│ │ ├── 头段
│ │ ├── 中段
│ │ └── 尾段
│ └── ECU
│ ├── 一阶程序
│ ├── 二阶程序
│ └── ...

text

**优势**：
- 可扩展性强：随时增加新类别
- 粒度自由：用户可从顶层一直钻到最细颗粒度
- 适合移动端：逐层展开，操作直观

## 二、顶层分类设计（一级分类）

知识库顶层按 **“系统部位”** 划分，共 **10 个一级分类**：

| 序号 | 一级分类 | 涵盖内容 |
| :---: | :--- | :--- |
| 1 | **动力系统** | 进气、排气、ECU、涡轮增压、冷却 |
| 2 | **底盘悬挂** | 避震器、弹簧、防倾杆、摆臂、衬套 |
| 3 | **轮毂轮胎** | 轮毂工艺/参数/品牌、轮胎分类/参数/品牌 |
| 4 | **刹车系统** | 刹车片、刹车盘、卡钳、刹车油管 |
| 5 | **外观改装** | 包围、宽体、尾翼、改色膜、灯光 |
| 6 | **内饰改装** | 方向盘、座椅、仪表、影音、氛围灯 |
| 7 | **车身强化** | 顶吧、底盘拉杆、防滚架 |
| 8 | **油水养护** | 机油、刹车油、变速箱油、冷却液 |
| 9 | **电子系统** | 仪表、行车记录仪、雷达、胎压监测 |
| 10 | **新手入门** | 改装顺序、避坑指南、合法备案、术语词典 |

七、内容填充优先级建议
优先级	分类	原因
P0（首发）	新手入门	吸引新手用户，降低门槛
P0（首发）	轮毂轮胎	新手最常问、最常改的
P0（首发）	底盘悬挂	“百改降为先”，需求量最大
P1（第二期）	动力系统	进阶用户需求
P1（第二期）	刹车系统	安全相关，必须覆盖
P2（后续）	外观/内饰/车身强化	个性化需求
P2（后续）	油水/电子	养护补充内容
附：维护与扩展建议
每新增一个改装件类别，先在对应一级分类下建立叶子节点，再补充内容。

品牌库独立维护，与内容通过 brand_id 关联，避免重复。

定期根据用户搜索热词，调整分类名称或新增别名，提高搜索命中率。

内容支持“标签”系统，可实现跨分类聚合（如 #绞牙 #日系 #性价比）。
